# AGENCY7 - Premium Digital Agency Website

A premium, highly interactive digital agency website built with HTML5, CSS3, and Vanilla JavaScript. 

## Features
- **Responsive Modern Design**: Elegant layout with a cohesive dark-blue and cream color palette.
- **Scroll-Reveal Animations**: Seamless fade-up and zoom transitions as you scroll, powered by a JavaScript `IntersectionObserver` with zero-config HTML markup.
- **Interactive Components**:
  - Vertical sliding services carousel.
  - Interactive tabs showing ads, results, and custom analytics.
  - Integrated testimonial slider.
  - Interactive contact form.
- **Optimized Assets**: Pre-cleaned and lightweight images/videos for fast load times.

## File Structure
- `index.html` - Homepage.
- `about.html` - About Us page.
- `contact.html` - Main contact/project intake page.
- `unanswered.html` - Dedicated landing page.
- `style.css` - Custom styling & animation systems.
- `script.js` - Dynamic interactions, sliders, and animation controller.
- `images/` - Optimized graphics, photos, and background videos.
